# Ops Triage Skill

## Priority Model

Classify items as:

- P1: urgent + high impact
- P2: important but not urgent
- P3: low impact / backlog

## Output Requirements

For each item include:

- priority
- owner
- deadline
- action type (reply/schedule/delegate/archive)
- rationale (one line)

## SLA Rule

- P1 acknowledged immediately
- P2 batched in next planning cycle
- P3 grouped for async cleanup
